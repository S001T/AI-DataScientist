# AI Data Scientist

AI-powered Data Analysis Framework for Python.

## Installation

```bash
pip install aidatascientist
```

## Quick Start

```python
from aidatascientist import DataAgent

agent = DataAgent()

agent.load("Iris.csv")

agent.clean()

stats = agent.analyze()

print(stats)
```

## Features

* Data Cleaning
* Statistics Analysis
* AutoML
* Data Visualization
* OpenAI Integration
* Claude Integration
* Dataset Chat Agent
* SQL Agent
* Dashboard
